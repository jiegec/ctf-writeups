# Redbud Puppeteer

[Puppeteer](https://thuctf.redbud.info/) is a weekly CTF event by Redbud. You can play the challenges in my [archive](https://ctf.jia.je/).

- Week 0:
    - [Easy Random 1](./week0/easy-random-1.md): Attack glibc RNG
    - [Easy Random 2](./week0/easy-random-2.md): Attack python RNG
    - [The Old Gods Are Whispering](./week0/the-old-gods-are-whispering.md): Audio steganography
- Week 1:
    - [Easy Pyjail 1](./week1/easy-pyjail-1.md): Simple pyjail banning builtins
    - [Easy Random 3](./week1/easy-random-3.md): Attack python RNG using gf2bv
    - [Easy RE 1](./week1/easy-re-1.md): Simple reversible cipher to RE
    - [Rosetta 3](./week1/rosetta-3.md): Three character encodings on dcode.fr
    - [RSA All in One](./week1/rsa-all-in-one.md): Five typical RSA attacks
