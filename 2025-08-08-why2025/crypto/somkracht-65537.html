Mathematical Analysis

Known values:

  • \(N = p \times q\)
  • \(e = 65537\)
  • \(ct1 = msg^e \bmod N\)
  • \(ct2 = msg^{p + q} \bmod N\)
  • \(msg^{(p-1)(q-1)} \bmod N = 1\) (Euler's theorem)

Simplification:

  • \(msg^{pq-p-q+1} \bmod N = 1\)
  • \(msg^{p+q} \bmod N = ct2\)
  • \(msg^{pq+1} \bmod N = ct2\)
  • \(msg^{N+1} \bmod N = ct2\)

Solution approach:

  • Find integers \(x\) and \(y\) such that \(65537x + (N + 1)y = 1\) using the extended Euclidean algorithm
  • Compute \(ct1^x \times ct2^y \bmod N = msg^{65537x + (N+1)y} \bmod N = msg \bmod N\)

from Cryptodome.Util.number import inverse, long_to_bytes

def extended_gcd(a, b):
    x0, x1, y0, y1 = 1, 0, 0, 1
    while b != 0:
        q, a, b = a // b, b, a % b
        x0, x1 = x1, x0 - q * x1
        y0, y1 = y1, y0 - q * y1
    return a, x0, y0
